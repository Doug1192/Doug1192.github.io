Black-Scholes Options Pricer — Technical Details

Project Overview

Comprehensive options pricing application implementing the Black-Scholes-Merton model for European options with real-time Greeks calculation and Monte Carlo simulation capabilities.

Key Features

  • Black-Scholes Pricing: Analytical solution for European call and put options
  • Greeks Calculator: Real-time Delta, Gamma, Theta, Vega, and Rho
  • Volatility Analysis: Implied volatility using Newton-Raphson method
  • Monte Carlo Simulation: 10,000+ path simulations for exotic options
  • Volatility Smile: Interactive visualization of volatility skew across strikes
  • Sensitivity Analysis: Heatmaps showing option value changes

Technical Stack

  • Python 3.9 with NumPy, SciPy
  • Streamlit for interactive web interface
  • Plotly for visualizations
  • Pandas for data handling

Results & Impact

  • 99.9% accuracy compared to Bloomberg terminal prices
  • Greeks validated against professional trading platforms
  • Monte Carlo convergence within 0.5% of analytical solutions
  • Processing time < 0.1 seconds for single option pricing

Loan Default Prediction — Technical Details

Project Overview

Machine learning model predicting loan default risk using ensemble techniques. Analyzes borrower characteristics, financial history, and loan attributes to provide probability scores.

Key Features

  • Feature Engineering: 50+ features including debt-to-income, credit utilization, payment history
  • Ensemble Modeling: XGBoost, Random Forest, Logistic Regression with weighted voting
  • Class Imbalance: SMOTE for synthetic minority oversampling
  • Model Interpretability: SHAP values for feature importance
  • Risk Segmentation: Low, Medium, High, Very High risk categorization
  • Production Pipeline: Automated end-to-end prediction workflow

Model Performance

  • Accuracy: 87% | ROC-AUC Score: 0.91
  • Precision: 83% | Recall: 79%
  • Dataset: 250,000+ loan applications

Top Predictive Features

  • Credit Score (FICO): 25% feature importance
  • Debt-to-Income Ratio: 18% importance

  • Credit Inquiries: 8% importance

Fama-French Analysis — Technical Details

Project Overview

Empirical analysis of Disney (DIS) and Amazon (AMZN) stock returns using the Fama-French 3-factor model with weekly data from 2013–2014.

Regression Results

Disney (DIS):

  • Market Beta: 1.095 (highly significant, p<0.0001)
  • SMB Factor: -0.373 (marginally sig, p=0.078) → Large-cap stock
  • HML Factor: -0.496 (marginally sig, p=0.067) → Growth stock
  • R²: 42.89%

Amazon (AMZN):

  • Market Beta: 1.092 (highly significant, p<0.0001)
  • SMB Factor: 0.471 (not sig, p=0.220) → Mid-cap tendencies
  • HML Factor: -1.268 (sig, p=0.011) → Strong growth stock
  • R²: 27.58%

Key Findings

  • Both stocks have similar market betas (~1.09)
  • Disney is large-cap growth, Amazon is mid-cap strong growth
  • Model explains Disney better (higher R²)
  • Amazon has more idiosyncratic risk
